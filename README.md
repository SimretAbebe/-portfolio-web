## 🛠️ Local Development

To run this portfolio project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SimretAbebe/simret-portfolio.git
   cd simret-portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
